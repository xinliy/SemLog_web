Welcome to SemlogWeb's documentation!
=====================================

**SemlogWeb** is a computer vision framework that bridges between Virtual Reality and perception algorithms.
It is designed to be distributed and efficient with the following advantages:

- User-friendly interface connected to Virtual Reality.
- Easy deployment to internet for concurrent access.
- Clear UI for image/point cloud dynamic visualization.
- Support of parallel and GPU learning.
